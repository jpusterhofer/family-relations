To run:
swipl family.pl

Queries:
nthcousinkremoved(X, Y, N, K).
kthchild(X, Y, K).

This program uses Prolog queries to determine two different relationships between family members.
nthcousinkremoved queries if X and Y are the Nth cousin K times removed.
kthchild queries if X is the Kth child of Y.
